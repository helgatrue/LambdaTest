# cypressTest

## Tests
- Test Scenario 1: Change range value from 15 to 95
- Test Scenario 2: Fill the form

## Test run

Install all the dependencies.

```sh
npm install cypress --save-dev
```

To run all tests

```sh
npx cypress run
```

To open all tests

```sh
npx cypress open
```

## To Run in lambdatest cloud environment
```sh
lambdatest-cypress run
```
